# PRD Template

## 1. Executive Summary

- What are we building?
- For whom?
- What problem does it solve?
- What impact should it create?

## 2. Problem Statement

- Who has the problem?
- What is the problem?
- Why is it painful?
- What evidence supports it?

## 3. Target Users

- Primary user
- Secondary user
- Key behaviors or constraints

## 4. Strategic Context

- Business goal
- Why now
- Relevant market or product context

## 5. Solution Overview

- Proposed solution
- Key user flow
- Major tradeoffs or constraints

## 6. Success Metrics

- Primary metric
- Supporting metrics
- Baseline and target

## 7. Requirements

- Must-have requirements
- Acceptance notes
- Edge cases or dependencies

## 8. Out of Scope

- Explicit exclusions

## 9. Risks and Dependencies

- Key risks
- Mitigations
- External or internal dependencies

## 10. Open Questions

- Questions that still need answers
