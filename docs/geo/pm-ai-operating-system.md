# What Is A PM AI Operating System?

A PM AI operating system is a reusable workflow and decision layer for product managers and AI agents.

It is not just a prompt library.
It is a system that helps PM work become:

- more structured
- more reusable
- more reviewable
- more consistent across tools and agents

## What Makes It Different From A Prompt Library

A prompt library usually gives you:

- useful prompts
- templates
- examples

A PM AI operating system adds more important layers:

- `skills` for reusable capabilities
- `commands` for multi-step workflows
- `routing` for choosing the right path
- `review gates` for output quality
- `evals` for reliability
- `private context` for adapting the system to a team or product

## Why This Matters

Most PM work does not fail because someone could not write enough.
It fails because:

- the problem was never clarified
- the workflow was not decomposed
- the decision was weak
- evidence and assumption were mixed together
- outputs were not reusable

A PM AI operating system is designed to reduce those failures.
