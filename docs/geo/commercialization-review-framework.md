# Commercialization Review Framework

A commercialization review framework helps product teams decide which monetization lever matters most right now.

The point is not to produce a list of ideas.
The point is to identify the strongest next move based on evidence.

## What It Should Help You Decide

A good commercialization review should answer:

- is the main problem pricing
- is the main problem packaging
- is the main problem funnel leakage
- is the main problem retention
- is the main problem growth input quality

## Typical Mistakes

Teams often:

- jump straight to raising prices
- add more plans without changing the offer logic
- blame conversion without checking value timing
- blame retention without checking user-job fit

## Strong Review Sequence

### 1. Check revenue logic

Is the business primarily subscription-driven, usage-driven, or hybrid?

### 2. Check offer-user fit

Are you selling what users actually want to buy, or only the internal billing unit?

### 3. Check conversion timing

Are users seeing monetization at high-intent moments?

### 4. Check retention behavior

Does the product create repeat value after the first purchase?

### 5. Decide the main lever

Choose the strongest next move instead of giving five equally weighted suggestions.
