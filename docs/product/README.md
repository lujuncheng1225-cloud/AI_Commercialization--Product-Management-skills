# Product Surface

This page exists for one reason:
help someone understand the product in under 3 minutes.

It should feel like a product, not a repository tour.

## The Product In One Sentence

PM AI Skill Toolkit is a judgment-first PM workspace.
It helps people form a reusable decision brief before they hand work to an agent.

## The Fastest Path

1. Open the [Interactive Toolkit](../pm-skills-interactive-course.html)
2. Pick one route:
   [Pricing](https://lujuncheng1225-cloud.github.io/AI_Commercialization--Product-Management-skills/pm-skills-interactive-course.html?scenario=pricing-strategy),
   [PRD](https://lujuncheng1225-cloud.github.io/AI_Commercialization--Product-Management-skills/pm-skills-interactive-course.html?scenario=prd-draft),
   [Monetization](https://lujuncheng1225-cloud.github.io/AI_Commercialization--Product-Management-skills/pm-skills-interactive-course.html?scenario=monetization)
3. Review the decision brief
4. Save it into `Brief Workspace` or copy a kickoff into Codex, Claude Code, or Cursor

## What The Surface Must Communicate

- This is not a prompt builder
- The first output is a decision brief
- The brief can be saved, reopened, duplicated, and compared
- The repo layer is optional until the user wants more depth
