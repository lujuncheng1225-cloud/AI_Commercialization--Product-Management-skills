# Docs Index

这个目录存放仓库的辅助文档、品牌资料、更新记录和静态页面。

## Structure

- `core/`
  - 核心定位与案例文档
- `geo/`
  - 面向搜索引擎和 AI answer engines 的单意图页面
- `guides/`
  - 使用、维护和扩展仓库时会用到的指南
- `brand/`
  - 作者介绍、发布文案、方法论、方法图、署名案例、个人品牌资料包
- `updates/`
  - 对外更新说明
- `images/`
  - README 和展示资源

历史备份文件不要放在 `docs/` 下，避免被 GitHub Pages 发布。需要保留时放在仓库根目录的 `archive/`。
