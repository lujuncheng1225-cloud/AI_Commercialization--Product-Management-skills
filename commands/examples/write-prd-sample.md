# Sample Output: Write PRD

## 1. Executive Summary

- 本项目目标是提升新用户 onboarding completion，通过减少首次设置阻力和缩短首个价值出现时间来提高激活率。

## 2. Problem Statement

- 新用户注册后需要完成多步配置，但在看到真实价值前已经流失。
- 当前流程更像资料收集，不像帮助用户快速完成任务。

## 3. Target Users

- 新注册的 self-serve 团队管理员
- 需要在首次登录后快速完成初始配置并邀请成员的用户

## 4. Success Metrics

- 主指标：onboarding completion rate
- 次指标：time to first value、invite completion rate
- Guardrails：客服工单量、设置失败率

## 5. Scope

- 简化首次设置步骤
- 重构引导文案和默认选项
- 优化首次成功路径

## 6. Out of Scope

- 不重做整套权限系统
- 不改邀请后的协作体验

## 7. Risks and Next Steps

- 风险：过度简化会导致后续配置缺失
- 下一步：拉齐设计稿、定义埋点、确认实验方案
