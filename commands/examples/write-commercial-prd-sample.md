# Sample Output: Write Commercial PRD

## 1. Executive Summary

- 本项目目标是提升自助升级转化，通过重做升级页、套餐表达和高意图购买触发点来提高收入。

## 2. Problem Statement

- 用户已获得可见价值，但在升级时无法快速理解哪种方案适合自己。
- 当前付费入口更像“买额度”，不像“完成作品”。

## 3. Target Segment

- 已完成至少一次可用 MV 的自助创作者
- 对高质量模型和导出有强需求的高意图用户

## 4. Success Metrics

- 主指标：upgrade conversion
- 次指标：首次付费客单价、购买前平均生成次数
- Guardrails：结果完成率、退款率

## 5. Proposed Changes

- 重写套餐价值表达
- 在结果页加入任务型购买推荐
- 优化额度不足时的升级引导

## 6. Risks

- 过度强调付费可能伤害免费体验
- 套餐改动可能造成老用户困惑

## 7. Rollout

- 先灰度到高意图分群
- 先验证结果页 uplift，再逐步扩大
