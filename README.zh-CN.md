# AI Product Management Skills

这是一个面向产品经理和 AI agent 的技能库。

它的目标不是只生成文档，而是把可复用的 PM 方法论编码成技能文件，让你以后提一个需求，就能更稳定地跑出结果。

## 仓库结构

- `skills/`：单个技能
- `commands/`：多技能串联工作流
- `docs/`：使用说明、构建说明、发布节奏
- `scripts/`：检索和 catalog 生成脚本
- `catalog/`：自动生成的技能索引

## 建议起步顺序

1. 读 `START_HERE.md` 或 `START_HERE.zh-CN.md`
2. 先跑一个技能，不要一开始就跑复杂工作流
3. 给足上下文：目标用户、业务指标、时间范围、约束

## AI PM 主题

这版已经覆盖了几类常见 AI PM 工作：

- 问题定义
- AI 功能定义
- LLM 评估方案
- 风险审查
- 模型选择
- 上线检查
- VOC 归纳
- Agent workflow 设计
